# Web UI Kit — El Rey

A marketing & admissions site for the chess school. One main surface (`index.html`) demonstrating header, hero, brand-values grid, programs, methodology, testimonial, contact form, and footer in a single editorial scroll.

## Components

- `Header.jsx` — sticky top nav with isotipo + section anchors, condenses on scroll.
- `Hero.jsx` — full-bleed verde-pino hero, oversized chess-piece decorative mark, eyebrow + display headline + lead + dual CTA.
- `ValuesGrid.jsx` — the six brand pillars (Cercanía, Pensamiento Estratégico, Crecimiento, Confianza, Formación, Acompañamiento), each as a small editorial card with its Material-Symbols icon.
- `Programs.jsx` — three programs by age (Iniciación, Intermedio, Avanzado) as tall feature cards.
- `Methodology.jsx` — split layout with quote on Verde Pino + step list on Crema.
- `Testimonial.jsx` — single-quote pull section in Malva for editorial warmth.
- `ContactForm.jsx` — inscription form with name/age/email/notes.
- `Footer.jsx` — calm crema footer with isotipo, address, links.

## Run

Open `index.html`. Babel transpiles JSX inline. No build needed.

## Notes & caveats

- The site is built from the brand guide first principles — there is no existing live site or Figma to mirror pixel-for-pixel. If one exists, share it and we will align this kit to match.
- Imagery slots use the brand's chess piece silhouettes as decorative placeholders. Replace with real photography (warm, soft-focus, human) when available.
